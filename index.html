laravel_session机制

laravel中处理Session的两种机制

  1. 全局的PHP辅助函数

  2. Request实例

  3. session中的基本方法

  • all() 获取session中的所有数据
  • has(键值) 是否存在某值，存在并且不为 null 的话返回 true
  • exists(键值) 是否存在某值，即使是 null 的话也无所谓 返回 true
  • put(‘key’, ‘value’) 存值
  • push(‘user.teams’, ‘developers’) 数组存储
  • pull(‘key’, ‘default’) 获取并删除数据
  • flush() 删除全部，重新生成
  • forget(‘key’); 移除键值
